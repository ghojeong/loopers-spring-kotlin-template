# Weekly I Learned 3 (Domain)

고객확인(KYC) 도메인을 딥다이브.
다음 개념들이 어떻게 객체로 구현되어 있고 서비스와 어떻게 연결되는지 도메인 용어와 객체설계를 파악.

회원인증 쪽은 기술 보다는 도메인 파악이 더 중요

- AML(Anti Money Laundary)
- KYC(Know Your Customer)
- EDD(Enhanced Due Diligence)
- CDD(Customer Due Diligence)
- RA(Risk Assessment)
- WLF(Watch List Filtering)
